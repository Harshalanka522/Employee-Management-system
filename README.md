# 👨‍💼 Employee Management System

A **Console-Based Employee Management System** developed using **Python** and **MySQL**. This project allows users to perform CRUD (Create, Read, Update, Delete) operations on employee records through a simple menu-driven interface.

---

## 🚀 Features

✅ Add Employee  
✅ View All Employees  
✅ Search Employee by ID  
✅ Update Employee Details  
✅ Delete Employee  
✅ MySQL Database Integration  
✅ Menu-Driven Console Application  

---

- 🐍 Python
- 🗄️ MySQL
- 🔗 mysql-connector-python

## 📌 How It Works

1️⃣ Connects to the MySQL database.

2️⃣ Displays a menu with employee management options.

3️⃣ Performs CRUD operations:
- ➕ Add Employee
- 📋 View Employees
- 🔍 Search Employee
- ✏️ Update Employee
- ❌ Delete Employee

4️⃣ Saves all changes to the MySQL database.

## ⚙️ Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/Harshalanka522/Employee-Management-system.git
```

### Navigate to the Project Folder

```bash
cd Employee-Management-system
```

### Install Required Package

```bash
pip install mysql-connector-python
```

### Configure Database

Update your MySQL credentials in `database.py` or `db_connection.py`.

```python
host = "localhost"
user = "root"
password = "YOUR_PASSWORD"
database = "employee_db"
```

### Run the Project

```bash
python main.py
```
